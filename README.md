# 个人起始页

## 项目概述

这是一个个人起始页网页前端项目，提供时间显示、搜索栏、应用图标网格和底部快捷栏等功能。项目采用纯HTML、CSS和JavaScript实现，不依赖任何框架，以实现轻量级、高性能的用户界面。

## 项目结构

```
项目根目录 (FY-next)
├── index.html          # 主HTML文件
├── styles.css          # 样式文件
├── script.js           # JavaScript脚本文件
├── background image/   # 背景图片目录
│   └── backgroud01.png # 背景图片
└── README.md           # 项目文档
```

## 主要功能

### 1. 时间显示
- 在页面顶部显示当前时间（小时:分钟）
- 使用JavaScript的`Date`对象每分钟更新一次

### 2. 搜索栏
- 提供搜索输入框
- 点击时有动画效果
- 支持多种搜索引擎选择

### 3. 应用图标网格
- 显示多个应用图标，排列成网格
- 点击时有动画效果
- 可显示/隐藏（通过底部快捷栏手机图标控制）

### 4. 底部快捷栏
- 在页面底部显示一排应用快捷图标
- 半透明背景和模糊效果
- 图标点击动画效果

### 5. 设置菜单
- 点击设置图标显示下拉菜单
- 提供主题切换、搜索引擎设置、壁纸更换等选项

## 技术特点

### 样式与交互
- **背景**：全屏背景图片，可自定义更换
- **字体**：使用'PingFang SC'和'Microsoft YaHei'等中文字体
- **颜色方案**：主要使用白色文字和图标，搭配半透明背景
- **CSS变量系统**：全面的CSS变量体系，包括颜色、字体大小、间距等
- **动画效果**：统一的过渡动画，提供流畅的用户体验

### 响应式设计
- **桌面端**：应用图标网格为5列
- **平板端**（≤768px）：应用图标网格调整为4列
- **移动端**（≤480px）：应用图标网格调整为3列

## 如何使用

1. **主题切换**：点击右上角设置图标(⚙️)，选择"页面主题"
2. **搜索引擎设置**：点击右上角设置图标(⚙️)，选择"搜索引擎设置"
3. **更换壁纸**：点击右上角设置图标(⚙️)，选择"更换壁纸"
4. **应用图标网格显示/隐藏**：点击底部快捷栏中的手机图标(📱)
