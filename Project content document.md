# 网页复刻项目文档

## 1. 项目概述

### 1.1 项目简介

本项目是一个网页前端复刻项目，旨在模仿特定UI界面，包含时间显示、搜索栏、应用图标网格和底部快捷栏等元素。项目采用纯HTML、CSS和JavaScript实现，不依赖任何框架，以实现轻量级、高性能的用户界面。

### 1.2 项目目标

- 精确复刻目标UI界面的视觉效果和交互体验
- 实现响应式设计，适配不同尺寸的设备
- 提供流畅的动画效果和用户交互
- 保持代码结构清晰，便于维护和扩展

## 3. 功能模块详解

### 3.1 时间显示模块

**功能描述**：在页面顶部显示当前时间（小时:分钟）。

**实现方式**：
- 使用JavaScript的`Date`对象获取当前时间
- 通过`setInterval`函数每分钟更新一次时间显示
- 使用CSS设置字体大小、颜色和阴影效果

**代码位置**：
- HTML: index.html中的`time-display`元素
- CSS: styles.css中的`.time-display`样式
- JS: script.js中的`updateTime`函数

### 3.2 搜索栏模块

**功能描述**：提供搜索输入框，点击时有动画效果。

**实现方式**：
- 使用HTML创建搜索栏结构，包含图标和输入框
- 通过CSS设置搜索栏的样式和动画效果
- 使用JavaScript监听点击事件，触发动画效果

**代码位置**：
- HTML: index.html中的`search-bar`元素
- CSS: styles.css中的`.search-bar`相关样式
- JS: script.js中搜索栏点击事件监听

### 3.3 应用图标网格模块

**功能描述**：显示多个应用图标，排列成网格，点击时有动画效果。

**实现方式**：
- 使用CSS Grid布局创建网格
- 为每个图标添加点击动画效果
- 响应式设计，在不同屏幕尺寸下调整网格列数

**代码位置**：
- HTML: index.html中的`app-grid`元素
- CSS: styles.css中的`.app-grid`和`.app-icon`相关样式
- JS: script.js中应用图标点击事件监听

### 3.4 底部快捷栏模块

**功能描述**：在页面底部显示一排应用快捷图标。

**实现方式**：
- 使用Flexbox布局创建水平排列的图标栏
- 添加半透明背景和模糊效果
- 为图标添加点击动画效果

**代码位置**：
- HTML: index.html中的`bottom-dock`元素
- CSS: styles.css中的`.bottom-dock`和`.dock-icon`相关样式
- JS: script.js中底部快捷栏图标点击事件监听

### 3.5 设置菜单模块

**功能描述**：点击设置图标显示下拉菜单，点击其他区域隐藏菜单。

**实现方式**：
- 使用HTML创建菜单结构
- 通过CSS控制菜单的显示和隐藏
- 使用JavaScript监听点击事件，切换菜单的显示状态

**代码位置**：
- HTML: index.html中的`settings-menu`元素
- CSS: styles.css中的`.settings-menu`相关样式
- JS: script.js中设置菜单显示/隐藏功能

## 4. 样式与交互

### 4.1 视觉风格

- **背景**：使用全屏背景图片，添加半透明黑色遮罩增强可读性
- **字体**：使用'PingFang SC'和'Microsoft YaHei'等中文字体
- **颜色方案**：主要使用白色文字和图标，搭配半透明背景
- **阴影效果**：为文字和元素添加阴影，增强立体感和可读性

### 4.2 动画效果

- **点击反馈**：图标点击时有缩放动画
- **搜索栏动画**：点击搜索栏时有放大效果
- **菜单过渡**：设置菜单显示/隐藏时有平滑过渡效果

### 4.3 响应式设计

- **桌面端**：应用图标网格为5列
- **平板端**（≤768px）：应用图标网格调整为4列，搜索栏宽度增加
- **移动端**（≤480px）：应用图标网格调整为3列，时间显示字体缩小
